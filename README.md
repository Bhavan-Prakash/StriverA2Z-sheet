# Striver A2Z Sheet Solutions

My solutions to the **Striver A2Z DSA Sheet**, implemented in **Java**. This repository is a personal collection of solved problems, organized topic-wise as I progress through the sheet.

**Striver A2Z DSA Sheet:** https://takeuforward.org/dsa/strivers-a2z-sheet-learn-dsa-a-to-z

## Repository Structure

```text
StriverA2Z-sheet/
│
├── Topic_Name/
│   ├── src/
│   │   ├── Problem_1.java
│   │   ├── Problem_2.java
│   │   └── ...
│   ├── out/
│   └── Topic_Name.iml
│
├── Another_Topic/
│   ├── src/
│   │   └── ...
│   └── ...
│
└── README.md
```

- Each folder represents a **topic** from the Striver A2Z Sheet.
- All solution files are inside the **`src/`** directory.
- Solutions are implemented in **Java**.
